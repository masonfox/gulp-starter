Welcome to Gulp Starter!

An out of the box Gulp build

What is Gulp-Starter?

Gulp-Starter is an out of the box gulp build that includes a plethora of modern web development tools. It is intended to excelerate project startup and provide consitency from project to project.

Never heard of Gulp?

Gulp is a toolkit that will help you automate painful or time-consuming tasks in your development workflow

This build includes

Bootstrap

The world's most popular responsive css framework! This repo is currently running version 3 of Bootstrap.

Browsersync

Browsersync not only injects instantant changes of css, js, and jade/html, but also across all devices on the same network! Say goodbye to manual reloading!

Lib Sass

Getting sassy with the fastest compiling SASS engine! Why be slow?

Autoprefixer

Hate vendor prefixes, don't worry about them anymore! Let autoprefixer take care of it. That's right! You write regular, non-prefixed declarations!

Bower

Who doesn't love a good ol' project package manager?

Jade Templating

Write more attractive and maintainable code with Jade! Never used it? It is super easy to use! Give it a shot!
